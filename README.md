# Edison Frade - Advocacia

## Sobre
Site estático para **Edison Frade**, advogado português-brasileiro especializado em:
- **Direito da Nacionalidade**: Naturalização, registo de nacionalidade portuguesa e brasileira
- **Direito Penal**: Defesa criminal e consultoria em matéria penal
- **Direito Criminal**: Procedimentos criminais e proteção de direitos fundamentais
- **Direito Familiar**: Casamento, divórcio, guarda de menores, pensão alimentícia e sucessão

## Características
- Design responsivo baseado em Bootstrap 4
- Navegação intuitiva com menu dropdown
- Páginas especializadas para cada área de atuação
- Blog para artigos jurídicos
- Depoimentos de clientes
- Formulário de contacto
- Mapa de localização integrado

## Páginas Principais
- `index.html` - Página inicial com apresentação geral
- `services.html` - Áreas de especialização detalhadas
- `about-us.html` - Informações sobre Edison Frade
- `blog.html` - Artigos jurídicos e notícias
- `contact.html` - Formulário de contacto e localização
- `testimonials.html` - Depoimentos de clientes
- `team.html` - Informações sobre a equipa
- `terms.html` - Termos e condições

## Estrutura de Pastas
```
/assets
  /css - Folhas de estilo (fontawesome, owl carousel, custom styles)
  /images - Imagens do site
  /js - Scripts JavaScript (custom, owl carousel)
/vendor
  /bootstrap - Framework Bootstrap
  /jquery - jQuery library
```

## Linguagem
O site está totalmente em português (português europeu com referências ao português brasileiro).
